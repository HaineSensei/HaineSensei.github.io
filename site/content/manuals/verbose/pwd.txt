pwd - Print working directory

USAGE:
  pwd

DESCRIPTION:
  Displays the absolute path of the current working directory.

EXAMPLES:
  pwd             Show current directory path

OUTPUT:
  The output shows the path from root, for example:
    /               (at root)
    /blog           (in blog directory)
    /blog/posts     (nested directory)

NOTES:
  - Useful for keeping track of your location while navigating
  - The path is always shown in absolute form from root
